# Platform_Runner
Unity 2019.4 LTS

User Manuel:

1- To play this game, click on the code, download the zip file.

2- Open or extract it wherever you want.

3- Under the Runner_Build folder, you will find Platform_Runner.exe file

4- Double click Platform_Runner.exe to run the game.


How to Play:

1- You can move the character by using w, a, s, d or arrow keys on your keyboard.

2- To make the character jump use the space key.

3- You need to avoid all the obstacles, and be careful with opponent, they can pusyh you into the obstacles.
